# 1. Product Overview
The Centralized Hotel Management Platform is a cloud-based web application that allows staff to manage reservations, check-ins, and room status across multiple properties from a single interface.

# 2. Target Users
- **Front Office Agent:** Needs to quickly create reservations and check guests in/out.
- **Corporate Manager:** Needs a dashboard showing occupancy and revenue across all 8 hotels.

# 3. Product Goals & Success Metrics
- **Metric 1:** 100% of reservations for Hotel 1 (Pilot) processed through the new system within 3 months of launch.

# 4. Features & Capabilities
## 4.1 Minimum Viable Product (MVP)
- **FEAT-001 [Multi-Property Selector]:** Users with corporate access can toggle their view between different properties. (Traces to BR-001).
- **FEAT-002 [Standardized Booking Engine]:** A unified interface to search availability and create reservations for a selected property. (Traces to BR-002).

## 4.2 Future Scope
- **FEAT-003 [Payment Gateway Integration]:** Direct credit card processing within the booking engine.

# 5. Assumptions & Open Questions
- [ASSUMPTION] Staff will access the platform via desktop web browsers, not mobile devices.

# PRD Review Guide

When reviewing a PRD, focus on:
1. **User Centricity:** Are the features solving real user problems?
2. **Alignment:** Does every feature trace back to a valid business requirement in the BRD?
3. **Prioritization:** Is the MVP scope realistic and clearly defined?
4. **Metrics:** Are the success metrics measurable and relevant to the product goals?

**Output Format for Review:**
- Executive Summary
- Critical Issues (e.g., orphan features, missing user personas)
- Missing Information
- Traceability Gaps
- Recommendations
